Feature 1: Exam Paper Upload API

Create an API that allows students or teachers to upload past exam papers.

Supported formats:

PDF

JPG

PNG

Example endpoint:

POST /api/upload-paper

Requirements:

Store uploaded files

Extract text from PDFs and images

Use OCR if necessary

Automatically detect questions from the exam paper

Suggested tools:

OCR engine

PDF parser

LLM API for question extraction

Feature 2: Question Extraction Engine

After upload, automatically extract questions and convert them into structured data.

The system should detect:

Question text

Multiple choice answers

Subject category

Correct answer (if possible)

Example structured output:

{
  "subject": "Accounting",
  "questions": [
    {
      "question": "What is a balance sheet?",
      "options": [
        "A financial statement",
        "A marketing report",
        "A sales document",
        "A cash receipt"
      ],
      "answer": "A financial statement"
    }
  ]
}

Store all parsed questions in a database so they can be reused.

Feature 3: Game Generation Engine

Feature 4: AI Tutor Mode

Create an AI tutoring feature that explains questions in simple terms for high school students.

Example endpoint:

POST /api/explain-question

Example response:

Question: What is a balance sheet?

Explanation:
A balance sheet shows the financial position of a company at a specific moment in time. It lists assets, liabilities, and equity.

The AI tutor should:

simplify explanations

give examples

connect concepts to real-world finance
